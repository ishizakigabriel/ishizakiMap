Sobre o autor

Gabriel Ishizaki (23 anos), estudante do Instituto Federal de São Paulo (IFSP) - Campus de Hortolândia, iniciou seus estudos na instituição no ano de 2017, após cursar desenvolvimento de Jogos Digitais. Tendo adquirido conhecimento prévio em programação, principalmente na linguagem C#, tem estudado desenvolvimento WEB utilizando HTML, CSS e Javascript, sob os cuidados do professor Bordignon, para aprimoração de sua carreira profissional.


O Projeto

Esse projeto tem como objetivo facilitar a navegação ao usuário para viabilizar a busca por praças, para divertimento em família, e centros esportivos públicos, para a prática esportiva.


Ferramentas

HTML

Hypertext Markup Language (HTML) é uma linguagem de marcação de hipertexto, sendo uma das principais ferramentas de desenvolvimento de páginas web.

CSS

Cascading Style Sheet (CSS), apesar de ser se assemelhar bastante a uma linguagem de programação, é apenas uma ferramenta de definição de estilos de uma página WEB, sendo utilizada essencialmente em personalização e estilização de um site.

Javascript

O Javascript (diferente do Java, sendo confundido ocasionalmente) é uma linguagem de programação comumente utilizada em conjunto com HTML e CSS, para definições lógicas e validação de conteúdo de uma página.

Leaflet

Uma biblioteca open source adicional para o javascript. Permite, através de programação, utilizar mapas globais em suas páginas, com marcações entre outros recursos.
